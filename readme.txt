== Description ==

This plug-in REQUIRES the qTransalte to work porperlly. It exports all your pages into a Excel2007 (id, titleES, titleEN, contentES, contentEN)
separating the languages, so you can transtale your whole site offline.

The actual supported HTML tags are the following (the rest will be striped OUT! )
 * <b></b>
 * <i></i>
 * <h1></h1>
 * <h2></h2>
 * [shorcodes]


== Installation ==

1. Upload `plugin-name.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
